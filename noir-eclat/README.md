An interactive, cinematic brand site for a fictional high-jewelry maison.

## Stack

- **Next.js 14** (App Router) + **TypeScript**
- **Three.js** ‚Äî real-time WebGL ring (metallic PBR, faceted gem, bloom, particles)
- **GSAP + ScrollTrigger** ‚Äî pinning, horizontal scroll, scrubbed reveals
- **Lenis** ‚Äî smooth scroll, synced to ScrollTrigger
- **Framer Motion** ‚Äî intro, masked text, micro-interactions
- **Tailwind CSS** ‚Äî design tokens, layout
- CSS `mask` / `mix-blend-mode` / `clip-path` / `backdrop-filter` for atmosphere

## The Three.js centerpiece

`components/three/RingCanvas.tsx` renders the hero & signature ring in real time:

- procedural ring: `TorusGeometry` band + faceted gem (`MeshPhysicalMaterial`,
  metalness/clearcoat/transmission) + prongs
- a **custom studio environment** (canvas ‚Üí `PMREMGenerator`) gives the metal
  bright sources to reflect ‚Äî a dark room with a luminous ceiling, gold rim and
  teal accent
- **UnrealBloom** post-processing for the sparkle
- mouse-tilt + scroll rotation + auto-spin, gold particle field
- pauses when off-screen (`IntersectionObserver`), caps DPR, disposes on unmount
- falls back to the PNG cutout under `prefers-reduced-motion` or no-WebGL

## Images

Placeholder art is generated by `scripts/generate-placeholders.mjs` into
`public/images/`. **Drop real photography in with the same filenames to swap
assets ‚Äî no code changes needed.** Paths are centralized in `lib/constants.ts`.

| File | Use |
| --- | --- |
| `ring-silver-cutout.png` | Hero / Signature fallback, collection |
| `ring-blue-gold-cutout.png` | Collection, viewing room |
| `ring-silver-macro.png` | Macro, viewing room |
| `ring-blue-gold-macro.png` | Macro detail (full-bleed) |
| `bg-gold-dust.png` | Hero / Manifesto atmosphere |
| `bg-black-marble.png` | Craftsmanship texture |

## Accessibility & performance

- full `prefers-reduced-motion` path (no smooth scroll, no reveals, PNG ring)
- horizontal scroll collapses to a vertical stack on mobile
- `transform`/`opacity`-driven animation, capped DPR, off-screen pause
- semantic `button`/`a`, visible keyboard focus, `aria-label`s on nav
